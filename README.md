# fgb-admin

A single-page application for fgb-api.

## Development

### Terminal 1

```
$ npm install
$ npm run watch
```

### Terminal 2

```
$ npm install -g json-server
$ json-server test/fixtures/db.json
```

## Configuration

name                           | value
-------------------------------|-------------------------------
API\_BASE\_URL                 | `http://localhost:3000`
